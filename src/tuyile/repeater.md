::: info 功能信息
**指令**: `无`  
**触发:** `自动`   
**支持:** `文本` / `贴纸`
<Badge text="指令映射❎"/> <Badge text="REPL模式❎"/>
:::

::: tip 功能说明
群组中, `消息或贴纸重复两次` / `消息以感叹号结尾` 即可触发复读  
如果消息是 `转发` 或 `回复`, bot也会同步
:::
