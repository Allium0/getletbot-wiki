::: info 功能信息
**指令**: `/col`  
**触发:** `指令` / `自动`   
**支持:** `文本`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明

| 参数名称                                  | 参数说明                      |
|---------------------------------------|---------------------------|
| `宽度`                                  | 默认 200                    |
| `高度`                                  | 默认 100                    |
| `颜色`                                  | RGB: 用逗号隔开 / HEX: #开头十六进制 |
| `方向`<Badge text="需要两个颜色" type="tip"/> | 文字 / 箭头 / 箭头emoji         |

参数顺序随意

<Badge text="指令示例:" type="tip"/>

- `/col 255,0,0`
- `/col #ff0000`
- `/col #ff0000 0,0,255`
- `/col #ff0000 0,0,255 ↑ `
- `/col #ff0000 0,0,255 ↓`
- `/col #ff0000 0,0,255 右下 100`
- `/col #ff0000 0,0,255 ↖️ 100 300`
  :::
