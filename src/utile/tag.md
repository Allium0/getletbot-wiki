---
title: 获取图片标签
icon: tags
---

::: info 功能信息
**指令**: `/tag`  
**触发:** `指令回复`   
**支持:** `贴纸` / `图片`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明
| 参数名称 | 参数说明 |
|------|------------------------|
| `w` | 使用 wd14 识别 |
| `a` | 使用 azure 识别 |
| `c` | 使用 azure 生成图片标题 |
| `d` | 使用 azure 生成密集图片标题 |

:::
