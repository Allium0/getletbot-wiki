::: info 功能信息
**指令**: `/upi`  
**触发:** `指令回复`   
**支持:** `图片` / `贴纸` / `GIF` / `文件`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明
一个基于IPFS的图床，图片将上传到IPFS网络上。  
支持涩图，可在任意IPFS节点上访问图片，  
图片在一段时间内没有访问会自动删除，无法主动删除图片  
:::
