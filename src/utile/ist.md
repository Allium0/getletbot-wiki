::: info 功能信息
**指令**: `/ist`  
**触发:** `指令回复`   
**支持:** `图片` / `贴纸` / `GIF`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明

| 参数名称 | 参数说明        |
|------|-------------|
| `r`  | 先去背景再转换     |
| `e`  | 转换成自定义Emoji |

可以绕过 Telegram 视频贴纸 3 秒限制

<Badge text="指令示例:" type="tip"/>
- `/ist r`  
- `/ist e`  
:::
