::: info 功能信息
**指令**: `/o`  
**触发:** `指令回复`   
**支持:** `文本` / `媒体`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明

| 参数名称     | 参数说明              |
|----------|-------------------|
| `±[1-9]` | 消息数量              |
| `背景色`    | 格式为十六进制，输入两个颜色为渐变 |
| `r`      | 不添加回复消息           |
| `用户id`   | 《无中生有》            |

参数顺序随意

<Badge text="指令示例:" type="tip"/>
- `/o 3`
- `/o r`
- `/o -2 #ffffff`
- `/o #ffffff/#000000 3`
- `/o #ffffff`
- `/o 1447514373 2 #ffffff`
:::
