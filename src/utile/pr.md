::: info 功能信息
**指令**: `/pr`  
**触发:** `指令回复` / `文件说明`  
**支持:** `图片` / `贴纸` / `GIF` / `视频`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明

不加参数默认为`动漫人物识别`

| 参数名称  | 参数说明    |
|-------|---------|
| `ani` | 动漫人物识别  |
| `gal` | Gal人物识别 |

<Badge text="指令示例:" type="tip"/>
- `/pr ani`  
- `/pr gal`  
:::
