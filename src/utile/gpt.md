::: info 功能信息
**指令**: `/gpt`  
**触发:** `指令` / `指令回复`   
**支持:** `文本` / `媒体` / `贴纸` / `GIF`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明
加上 `/gs` 可以触发谷歌搜索  
可以给搜索添加一个[指令映射](../system/map.md)方便使用

暂不支持上下文, 后续会加
<Badge text="指令示例:" type="tip"/>

- `/gpt /gs北京天气`

:::
