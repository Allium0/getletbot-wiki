::: info 功能信息
**指令**: `/sd`  
**触发:** `指令回复`   
**支持:** `贴纸`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明
下载单个动态贴纸支持转为GIF  
下载整个动态贴纸包不支持转换为GIF,可以自行用在线网站转换

为什么下载整个包不支持转换为GIF?  
费性能. 而且一般也用不了整个包, 下载自己需要的贴纸即可  
推荐使用 [REPL 模式](../system/repl.md)
:::
