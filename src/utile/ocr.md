::: info 功能信息
**指令**: `/ocr`  
**触发:** `指令回复`   
**支持:** `图片` / `贴纸`
<Badge text="指令映射✅"/> <Badge text="REPL模式✅"/>
:::

::: tip 功能说明

| 参数名称  | 参数说明     |
|-------|----------|
| `t`   | 识别并微软翻译  |
| `gpt` | 识别并GPT翻译 |
| `yd`  | 识别并有道翻译  |
| `tts` | 识别并生成语音  |

<Badge text="指令示例:" type="tip"/>
- `/ocr t`  
- `/ocr tts`  

:::
