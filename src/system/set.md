::: info 功能信息
**指令**: `/set`  
**触发:** `指令`   
**支持:** `文本`
<Badge text="指令映射❎"/> <Badge text="REPL模式❎"/>
:::

::: tip 功能说明
Bot的所有功能都可在管理面板中配置，命令列表也会同步隐藏 / 显示指令  
`/set 关键词` 可以查找功能

群内只有管理员可用, 可配置`群管功能`
:::
